<details> <summary><sub>⚙️ Snake animation not showing? Click here for the 2-minute fix</sub></summary> <br/>
The snake needs a one-time setup because it reads your actual contribution graph:

Create a repo named exactly mankirat657 (must match your username — this is your GitHub profile repo).

Inside it, add the included snake.yml file at the path .github/workflows/snake.yml.

Go to Settings → Actions → General → Workflow permissions and select "Read and write permissions."

Go to the Actions tab, select "Generate Snake Animation," and click "Run workflow" to trigger it manually the first time.

Wait ~1 minute — it creates an output branch with your snake SVGs. Refresh your profile page and it'll appear.

After that, it regenerates automatically every day.
